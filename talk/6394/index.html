Vue Dojo - 實作 Vue.js 專屬的線上編輯器

Peter Chen (皮特)

研華科技

網頁的開發者或多或少都會使用到 CodePen 或是 JSFiddle 這類的線上代碼編輯器，這些神奇的服務在網頁上就可以做一般 IDE 大部分的工作，對於 DEMO ，練習，實驗時有很大的助益。

本次會介紹線上的編輯器是如何運作的，並且以 Vue.js 做進一步的優化，增加對 Vue.js Single File Component 的支援，並做編譯將其結果顯示於網頁上。
