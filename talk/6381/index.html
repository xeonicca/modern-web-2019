完全用 JavaScript 構築 UI 架構體系

A  101AB  16:20:00 - 17:00:00

程劭非 (winter)

Flutter 的出現，給我們帶來了全新的 UI 架構思路，繞開作業系統的 UI 體系，應用 Dart 語言從 Skia 層直接構築 UI 體系，完美實現了跨平臺開發，這樣使得 UI 體系對使用者更容易擴展和調試。

於是我不禁在思考，群眾基礎更好，從一出生就跟 UI 聯繫在一起的 JavaScript 是否也能做到同樣的事情呢？於是，在最近一年裡，我開始在社區參與SpriteJS 專案，跟他的作者一起把它變成一個完整的 UI 系統。

About the speaker

曾任職於微軟亞洲工程院（瀏覽器開發）、盛大創新院（電子書和 WebOS 開發）和手機淘寶（前端組 Leader、終端架構組 Leader），在 QCon、Top100、FEDay、JSConf 等技術會議有多次分享和出品人經歷，也是 ModernWeb 第二屆的演講者和第三屆的出品人，現在辭職專門從事程式設計教育工作，致力於給行業帶來更多優秀人才。
