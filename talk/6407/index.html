初探 Javascript 模組化開發 - 從 open source 取經

A  101AB  13:30:00 - 14:10:00

劉艾霖 (Alin)

Rooit

「模組化」一個熟悉又陌生的領域，多數的人都使用過框架或函式庫，但說到自行開發模組並公開發佈讓別人使用，則佔少數。

將分享講者本身參與 open source 開發，從中學到如何透過模組方式協同開發，並將這樣的經驗，擴展至其他地方。

About the speaker

大多的時間負責後端程式開發，熟悉 Javascript 與 Web 相關技術，熱衷鑽研各項技術研究、經驗分享、培訓授課與在台灣推廣遠距工作理念，是「遠距工作者在台灣」社團的創辦人。
