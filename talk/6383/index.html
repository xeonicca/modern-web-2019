郝稼力 (HiWebGL)

HiWebGL

自從 WebGL 與 HTML5 並肩作戰，共同為互聯網帶來新的特性，已經過去了十年。儘管成功為前端從業者帶來技術的進步和變革，但在網頁的世界中 WebGL 依然行如路人。如今，震天的戰鼓再一次響起！

本主題會建議前端從業者嘗試在自己的工作中使用 WebGL 帶來的 3D 特性，並將為聽眾展示當下流行和非常適用於 3D 的不同展示用途，如使用 WebGL Shader 代替 Javascript 和視頻特效等，並將解開大家心中對於 WebGL 易用性和性能問題的謎團，介紹當下已經趨於成熟的 Web 3D  生態，並將對網頁圖形的未來進行展望。
