初學 Go 專案目錄結構與設計

D  103  16:20:00 - 17:00:00

吳柏毅 (appleboy)

聯發科技

當開發者第一次接觸 Go 語言時，發現在 Go 語言沒有像其他語言有個非常活躍的框架可以追尋，像是 PHP Laravel 或 Ruby On Rails，所以初學者變得在學 Go 語言不知道該如何編排目錄結構。

本議程會帶給大家我在團隊內制訂的目錄結構及整合性測試，並藉由此議程來介紹團隊內使用了哪些 Go 套件，結合 CI/CD 流程做自動化測試及部署。

About the speaker

喜歡嘗試任何 DevOps 工具來改善團隊合作模式，省下更多時間來嘗試及學習新技術。

長期貢獻於 Open Source 專案，熱愛開發程式，目前專注於 Go 語言發展。曾任 PHPConf、JSDC、COSCUP 研討會講師，維護數個 Go 語言開源專案。
