How Modern Web Frameworks Boost Your Efficiency

B  101CD  13:30:00 - 14:10:00

蔣豪群 (soda)

Vue Technology LLC

現如今，流行的前端框架之間，為提高開發者生產力所進行的競爭，已經遠不止於「如何讓使用者寫好一個頁面/元件」了，而是包含了編譯優化、編輯器支援、瀏覽器調試工具等全方位的開發體驗上的支援和創新。

本次分享將結合分享者維護 Vue.js 相關工程設施的經驗，從框架本身的設計抽象、性能調優，到最終應用的開發調試體驗等方面，為聽眾展示一個典型的前端框架，是如何為開發者的工作效率做各種細緻優化的，並探討其中的一些設計權衡。

About the speaker

開源項目 Vue.js 核心團隊成員，曾就職於阿里巴巴，2018 年起成為 Vue.js 團隊（除作者尤雨溪以外）唯一的全職開源開發者。目前在團隊內負責 Vue CLI 和相關的 toolchain 維護。
