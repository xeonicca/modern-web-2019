開發現代化的靜態 PWA

黃胤翔 (Ben)

SHOPLINE

我們使用 React 開發過無數個網站，但每當建立新專案時，常需要花很多時間搞定開發環境、打包工具、解決 SEO 等問題 ，亦或是前端工程師們在接案時，想寫最少的後端語言、搭配某個開源 CMS 趕快結案拿錢，這時便很適合透過靜態網站來滿足我們的需求。

而作為一個 Static Site Generator，我們可以藉由 Gatsby 開發出帶有動態特性的靜態網站、寫著熟悉的 React、搭配最新的現代化工具，幫助我們簡單又快速產生 PWA。在這次分享中我將分析 Gatsby 的優劣勢、講解功能核心，並說明自己使用 Gatsby 開發時所遇到的地雷與其解決方案。
