Journey to Modern Web

C  102  15:15:00 - 15:55:00

張嘉顯 (Fred)

IBM

去年開始，我與部分成員決定啟動一個 side project，Apollo program，來幫助團隊朝向 Modern Web 更進一步。

我們的產品開發了七年以上，但前端技術在這幾年的時間當中，經歷了很大的變化，在不可能重寫整個專案的情況下，我們只能逐步的調整這個專案。我們使用約半年的時間，與其他幾位跨國成員利用自己的時間空擋，將 Webpack 導入專案。

我希望能夠分享我們做這件事情的經驗，與大家分享老舊專案如何逐步向現在的前端技術靠近。包括當初的考量、如何合作、時程的規劃、實際做下去遇到的問題和最後的結果。
 

About the speaker

誤打誤撞多摸了前端一點，但實際上熱愛各種技術，非常享受在思考、解決問題的過程當中。
